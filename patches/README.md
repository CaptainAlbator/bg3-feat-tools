# Metamagic Merged — Duplicate Patches

Optional patches that lock redundant Metamagic Adept feats from Essential Feats and/or Feats Extra.

## How It Works

These patches set a Charisma 30 requirement on the redundant Metamagic Adept entries. Since no character can naturally reach Charisma 30, the duplicates show a lock icon and cannot be selected. The actual Metamagic Adept from Lumaterian's Metamagic Enhanced (LHB_MetamagicAdept) remains fully functional.

This approach preserves save compatibility — the feats still exist in game data, they are just made inaccessible.

## Variants

| Patch | Locks | Requires |
|-------|-------|----------|
| **Duplicate Patch (Essential Feats)** | SYR_MetamagicAdept | ABCs of Feats + Metamagic Merged + Essential Feats |
| **Duplicate Patch (Feats Extra)** | MetamagicAdept | ABCs of Feats + Metamagic Merged + Feats Extra |
| **Duplicate Patch (Both)** | Both of the above | ABCs of Feats + Metamagic Merged + Essential Feats + Feats Extra |

**All three share UUID `c4d5e6f7-8901-2345-abcd-ef6789012345` — they are mutually exclusive. Install only one.**

## Load Order

Load the Duplicate Patch **after** Metamagic Merged:

1. Sorcerer class/subclass mods (if any)
2. Essential Feats and/or Feats Extra
3. Metamagic Enhanced
4. Metamagic Extended
5. Metamagic Merged
6. **Duplicate Patch** (your chosen version)

ABCs of Feats is an override mod — no load order needed, just place it in your Mods folder.

## Note on Using Both Feat Mods

Using Essential Feats and Feats Extra together results in many duplicate feats beyond just Metamagic Adept. The "Both" patch only handles the metamagic-related duplicates.

## Generation

The patches are generated by `metamagic_merged_duplicates_patch.py`.
